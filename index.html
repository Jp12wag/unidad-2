102 Etiquetas HTML

Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
<a> Define un hipervínculo. Se utiliza para enlazar una página a otra o un recurso. <a href="https://www.example.com">Visita Ejemplo</a> Se puede agregar el atributo target="_blank" para abrir el enlace en una nueva pestaña.
<p> Define un párrafo de texto. <p>Este es un párrafo.</p> Puede contener otros elementos en línea como <a>, <strong>, <em>.
<img> Incrusta una imagen en la página. <img src="imagen.jpg" alt="Descripción de la imagen"> Importante incluir el atributo alt para accesibilidad.
<div> Define una sección o contenedor en el documento. <div>Contenido del div</div> Usualmente usado para aplicar estilos o agrupar otros elementos.
<h1> Define el encabezado más importante en la página. <h1>Este es un Encabezado</h1> Los encabezados van desde <h1> (más importante) hasta <h6> (menos importante).
<h2> Define un subtítulo o un encabezado de menor importancia que <h1>. <h2>Este es un subtítulo</h2> Similar a <h1>, pero menos importante.
<h3> Define un encabezado de nivel 3. <h3>Encabezado de nivel 3</h3> Se utiliza para dividir contenido en secciones menores.
<h4> Define un encabezado de nivel 4. <h4>Encabezado de nivel 4</h4> Menos importante que <h3>.
<h5> Define un encabezado de nivel 5. <h5>Encabezado de nivel 5</h5> Menos importante que <h4>.
<h6> Define un encabezado de nivel 6. <h6>Encabezado de nivel 6</h6> El encabezado de menor importancia.
<strong> Resalta el texto con importancia fuerte. <strong>Texto resaltado</strong> El texto aparece en negrita.
<em> Resalta el texto con énfasis. <em>Texto con énfasis</em> El texto aparece en cursiva.
<span> Contenedor en línea para agrupar texto o elementos para aplicar estilos. <span style="color:red;">Texto rojo</span> Utilizado para aplicar estilos específicos en línea.
<br> Inserta un salto de línea. Primera línea<br>Segunda línea No requiere una etiqueta de cierre.
<hr> Inserta una línea horizontal para separar contenido. Contenido antes<hr>Contenido después No requiere una etiqueta de cierre.
<ul> Define una lista no ordenada. <ul><li>Elemento 1</li><li>Elemento 2</li></ul> Los elementos de la lista son precedidos por viñetas.
<ol> Define una lista ordenada. <ol><li>Elemento 1</li><li>Elemento 2</li></ol> Los elementos de la lista están numerados.
<li> Define un elemento de una lista. <ul><li>Elemento de lista</li></ul> Se utiliza dentro de <ul> o <ol>.
<table> Define una tabla en el documento. <table><tr><td>Celda</td></tr></table> Utilizada para organizar datos en filas y columnas.
<tr> Define una fila en una tabla. <tr><td>Celda 1</td><td>Celda 2</td></tr> Debe estar dentro de <table>.
<td> Define una celda en una tabla. <tr><td>Celda</td></tr> Debe estar dentro de <tr>.
<th> Define un encabezado de una celda en una tabla. <th>Encabezado</th> El contenido suele aparecer en negrita y centrado.
<thead> Define un grupo de filas de encabezado en una tabla. <thead><tr><th>Encabezado</th></tr></thead> Agrupa los encabezados en tablas complejas.
<tbody> Define el cuerpo principal de una tabla. <tbody><tr><td>Datos</td></tr></tbody> Separa el cuerpo de la tabla de su encabezado y pie de página.
<tfoot> Define un grupo de filas de pie de página en una tabla. <tfoot><tr><td>Pie de página</td></tr></tfoot> Utilizado para agregar un resumen o totales.
<form> Define un formulario para la entrada de datos. <form action="/submit"><input type="text"></form> Debe contener elementos de entrada como <input>.
<input> Define un control de entrada de datos. <input type="text" name="nombre"> Puedes especificar diferentes tipos como text, password, email, etc.
<textarea> Define un área de texto multilinea. <textarea>Texto aquí</textarea> Utilizada para entrada de datos largos.
<button> Define un botón clickable. <button>Enviar</button> Puedes usar type="submit" para enviar un formulario.
<select> Define un menú desplegable. <select><option>Opción 1</option></select> Los elementos <option> definen las opciones del menú.
<option> Define una opción en un menú desplegable. <option value="1">Opción 1</option> Utilizado dentro de <select>.
<label> Define una etiqueta para un control de formulario. <label for="nombre">Nombre:</label><input id="nombre" type="text"> Mejora la accesibilidad al asociar etiquetas con controles.
<fieldset> Agrupa controles de formulario relacionados. <fieldset><legend>Información Personal</legend></fieldset> Puede contener un <legend> para describir el grupo.
<legend> Define un título para un grupo de controles dentro de un <fieldset>. <fieldset><legend>Información Personal</legend><input type="text"></fieldset> Mejora la organización y legibilidad de formularios complejos.
<header> Define un encabezado para un documento o una sección. <header>Este es el encabezado</header> Puede contener elementos como logos, menús de navegación, etc.
<footer> Define el pie de página para un documento o una sección. <footer>Este es el pie de página</footer> Puede contener información como autor, derechos de autor, enlaces.
<article> Define un contenido independiente, como una publicación de blog o un artículo de noticias. <article>Contenido del artículo</article> Suele estar acompañado por un <header> y <footer>.
<section> Define una sección en un documento. <section>Contenido de la sección</section> Útil para dividir contenido en partes temáticas.
<nav> Define un conjunto de enlaces de navegación. <nav><a href="#home">Inicio</a><a href="#about">Sobre Nosotros</a></nav> Debe contener solo enlaces de navegación.
<aside> Define contenido adicional que se sitúa al margen del contenido principal. <aside>Este es un contenido adicional.</aside> Usualmente se coloca en barras laterales.
<main> Define el contenido principal del documento. <main>Este es el contenido principal</main> Solo debe haber un <main> por documento.
<figure> Define contenido ilustrativo, como una imagen, gráfico o código. <figure><img src="imagen.jpg" alt="Descripción"><figcaption>Descripción</figcaption></figure> Suele ir acompañado de un <figcaption>.
<figcaption> Proporciona una leyenda o descripción para el contenido dentro de un <figure>. <figcaption>Descripción de la imagen</figcaption> Debe estar dentro de un <figure>.
<audio> Incrusta un contenido de audio en el documento. <audio controls><source src="audio.mp3" type="audio/mpeg"></audio> Incluye controles de reproducción.
<video> Incrusta un contenido de video en el documento. <video width="320" height="240" controls><source src="video.mp4" type="video/mp4"></video> Puede incluir subtítulos mediante la etiqueta <track>.
<canvas> Proporciona un área de dibujo en la cual se puede renderizar gráficos mediante scripts. <canvas id="miCanvas"></canvas> Se utiliza con JavaScript para dibujar gráficos dinámicos.
<svg> Define gráficos vectoriales escalables. <svg width="100" height="100"><circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" /></svg> Permite definir figuras geométricas y gráficos 2D.
<iframe> Incrusta otro documento HTML dentro del documento actual. <iframe src="pagina.html"></iframe> Se usa comúnmente para incrustar vídeos, mapas, o contenido externo.
<meta> Define metadatos sobre el documento. <meta charset="UTF-8"> Se utiliza para definir el juego de caracteres, palabras clave, autor, etc.
<link> Define la relación entre el documento actual y un recurso externo. <link rel="stylesheet" href="styles.css"> Comúnmente se usa para vincular hojas de estilo CSS.
<style> Define estilos CSS para el documento. <style>body { font-family: Arial; }</style> Contiene reglas CSS que se aplican al documento HTML.
<script> Define un script de programación, generalmente JavaScript. <script src="script.js"></script> Puedes escribir scripts directamente o referenciarlos mediante un archivo externo.
<noscript> Proporciona contenido alternativo para usuarios que tienen JavaScript desactivado. <noscript>Por favor, habilita JavaScript para ver este contenido.</noscript> Es útil para mostrar mensajes o contenido alternativo.
<data> Asocia un contenido con un valor determinado. <data value="12345">Código: 12345</data> Permite asociar datos a un texto.
<datalist> Proporciona una lista de opciones predefinidas para un control de entrada. <input list="opciones"><datalist id="opciones"><option value="Opción 1"></datalist> Utilizado con el atributo list en <input>.
<dialog> Define un cuadro de diálogo o ventana emergente. <dialog open>Este es un diálogo</dialog> Puede incluir botones para aceptar o cancelar.
<menu> Define un grupo de comandos o un menú. <menu><li>Comando 1</li></menu> Frecuentemente se usa para crear menús contextuales.
<menuitem> Define un comando en un <menu> que el usuario puede invocar. <menuitem>Comando 1</menuitem> Este elemento ya no es compatible con HTML5.
<meter> Representa una medición escalar dentro de un rango conocido, como el nivel de batería. <meter min="0" max="100" value="50">50%</meter> Útil para indicar niveles de progreso o rendimiento.
<progress> Representa el progreso de una tarea en curso. <progress value="70" max="100">70%</progress> Se usa para mostrar el estado de una operación en curso.
<summary> Define un encabezado visible para un elemento <details>. <summary>Detalles</summary> Al hacer clic en el <summary>, se despliega el contenido de <details>.
<details> Define detalles adicionales que el usuario puede ver u ocultar. <details><summary>Más información</summary>Este es el contenido oculto</details> Es útil para crear secciones expandibles o colapsables.
<time> Representa un valor de fecha u hora. <time datetime="2024-09-10">10 de Septiembre de 2024</time> Mejora la semántica temporal en documentos.
<mark> Resalta texto con importancia o relevancia. <p>Este es un texto <mark>importante</mark></p> Útil para destacar términos clave en un texto.
<ruby> Proporciona anotaciones de texto en idiomas como el japonés o chino. <ruby>漢<rt>kan</rt>字<rt>ji</rt></ruby> Contiene elementos <rt> para definir la pronunciación.
<rt> Define la anotación o pronunciación en un texto <ruby>. <ruby>漢<rt>kan</rt>字<rt>ji</rt></ruby> Se usa dentro de <ruby> para mostrar la lectura.
<rbc> Define el contenedor de texto base en una anotación ruby. <ruby><rbc>漢字</rbc></ruby> Es parte de la estructura de anotaciones ruby.
<rp> Proporciona contenido adicional para navegadores que no soportan <ruby>. <ruby>漢<rp>(</rp><rt>kan</rt><rp>)</rp></ruby> Mejora la compatibilidad de <ruby> en navegadores antiguos.
<bdi> Permite aislar una porción de texto que podría cambiar la dirección del texto. <bdi>Texto de derecha a izquierda</bdi> Es útil para manejar textos que tienen diferente dirección de escritura.
<bdo> Anula la dirección del texto, permitiendo un control específico. <bdo dir="rtl">Texto en dirección contraria</bdo> Útil para escribir textos en direcciones opuestas, como en árabe o hebreo.
<wbr> Indica un posible punto de quiebre de línea en el texto. <p>Este es un texto con un <wbr>quiebre de línea</p> Útil para controlar cómo se visualizan los textos largos.
<ins> Define un texto que ha sido insertado en el documento. <p>Este es un <ins>texto insertado</ins> en el párrafo.</p> Por lo general, se muestra subrayado.
<del> Define un texto que ha sido eliminado del documento. <p>Este es un <del>texto eliminado</del> en el párrafo.</p> Por lo general, se muestra tachado.
<template> Define un fragmento de contenido HTML que puede ser clonado e insertado en el documento. <template><p>Este es un contenido de plantilla.</p></template> No se renderiza hasta ser insertado dinámicamente en el DOM.
<slot> Define un espacio donde se puede insertar contenido en componentes web personalizados. <slot name="header"></slot> Utilizado en componentes web para distribuir contenido.
<picture> Define un contenedor para varios elementos <source> y una imagen de respaldo. <picture><source srcset="imagen.webp" type="image/webp"><img src="imagen.jpg" alt="Descripción"></picture> Permite elegir diferentes imágenes según las capacidades del navegador.
<source> Define múltiples recursos multimedia para elementos <picture>, <audio> o <video>. <source src="video.mp4" type="video/mp4"> Permite especificar recursos alternativos para diferentes tipos o formatos de contenido.
<track> Define pistas de texto adicionales para elementos <audio> y <video>. <track kind="subtitles" src="subtitulos.vtt" srclang="es" label="Español"> Útil para subtítulos, descripciones de audio, y más.
<caption> Proporciona un título o descripción para una tabla. <table><caption>Título de la tabla</caption></table> Especifica la descripción de la tabla, generalmente en la parte superior.
<col> Especifica propiedades de estilo para una o más columnas en una tabla. <col span="2" style="background-color: lightblue"> Útil para aplicar estilos a columnas específicas de una tabla.
<colgroup> Define un grupo de una o más columnas dentro de una tabla para aplicarle estilos. <colgroup><col style="background-color: red"></colgroup> Se usa en combinación con <col> para agrupar columnas con estilos comunes.
<thead> Agrupa el contenido del encabezado de una tabla. <thead><tr><th>Encabezado</th></tr></thead> Útil para definir el encabezado de una tabla con varias filas de datos.
<tbody> Agrupa el contenido del cuerpo de una tabla. <tbody><tr><td>Contenido</td></tr></tbody> Define la sección del cuerpo de la tabla, generalmente debajo de <thead>.
<tfoot> Define un pie de tabla con información resumida o cálculos. <tfoot><tr><td>Resumen</td></tr></tfoot> Generalmente se coloca después de <tbody> para resumir los datos de la tabla.
<th> Define una celda de encabezado en una tabla. <th>Encabezado</th> Generalmente se muestra en negrita y centrado.
<optgroup> Agrupa opciones dentro de un <select>. <optgroup label="Grupo 1"><option>Opción 1</option></optgroup> Permite organizar opciones dentro de un menú desplegable.
<fieldset> Agrupa elementos relacionados en un formulario. <fieldset><legend>Información Personal</legend><input type="text" name="nombre"></fieldset> Generalmente utilizado con <legend> para etiquetar el grupo.
<legend> Proporciona un título o leyenda para un elemento <fieldset>. <legend>Información Personal</legend> Se coloca dentro de <fieldset> para describir el contenido agrupado.
<output> Representa el resultado de un cálculo o acción del usuario. <output name="resultado">0</output> Se puede usar con scripts para mostrar resultados de operaciones.
<embed> Incrusta un contenido externo como un archivo multimedia o una aplicación. <embed src="archivo.pdf" type="application/pdf" width="600" height="500"> Se usa para incrustar aplicaciones o contenidos externos en un documento HTML.
